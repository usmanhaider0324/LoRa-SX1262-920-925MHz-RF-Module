# LoRa SX1262 920–925MHz Long-Range RF Module

## 📌 Overview
This repository contains the complete hardware design of a 920–925MHz LoRa RF module based on the SX1262 transceiver. The design includes RF matching network optimization, controlled impedance routing, and external RF switch integration for long-range wireless communication.

The module is intended for ISM band applications requiring reliable low-power long-range communication.

---

## 🛠 Key Features
- SX1262 LoRa Transceiver
- 920–925MHz ISM Band Operation
- Optimized RF Matching Network
- Return Loss S11 < -37 dB @ 920–925MHz
- External RF Switch (PE4259)
- SMA Connector Output
- SPI Interface (MOSI, MISO, SCK, NSS)
- Dedicated RF Grounding Strategy

---

## 📊 RF Performance
The RF matching network was tuned for optimal impedance matching within the 920–925MHz band.

- Measured Return Loss (S11): < -37 dB
- Stable operation across the target frequency band
